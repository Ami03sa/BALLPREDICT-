# MVP to Production Roadmap

## MVP

- live game dashboard for one active game
- quarter, halftime, and final projections
- player momentum and defensive pressure indicators
- tactical explanation cards
- manual simulation controls for pregame what-if runs
- one historical training pipeline for core player box-score stats

## Phase 2

- real ingestion from NBA live and stats endpoints
- Redis-backed WebSocket fanout
- lineup-aware adjustments and substitution logic
- persistent projection history and "why did this change?" timeline
- user-selected strategy toggles like switching schemes or pace control

## Phase 3

- playoff series simulator
- richer shot chart and defender-location visualizations
- calibration dashboards and model registry
- coach/team-specific tactical fingerprints
- premium insight summaries and bet/DFS-oriented variants if desired

## Phase 4

- multi-game slate tracking
- automated retraining pipelines
- mobile notifications on projection swings
- custom lineup editor with possession-level simulation
- enterprise-grade observability and incident tooling
